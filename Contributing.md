# Contribuindo para o PHP Para Iniciantes

Obrigado por dedicar seu tempo para contribuir com este projeto!

Nosso objetivo é criar um material gratuito e de alta qualidade para estudantes de PHP.

---

## Como contribuir

Você pode colaborar de diversas maneiras:

* Corrigindo erros de digitação
* Melhorando exemplos
* Corrigindo bugs
* Adicionando exercícios
* Criando novos projetos
* Melhorando a documentação

---

## Antes de enviar um Pull Request

1. Faça um Fork do projeto.

2. Crie uma nova Branch.

```
git checkout -b minha-contribuicao
```

3. Faça suas alterações.

4. Realize seus commits.

```
git commit -m "Adiciona novo exercício sobre arrays"
```

5. Envie sua Branch.

```
git push origin minha-contribuicao
```

6. Abra um Pull Request.

---

## Boas práticas

* Escreva código limpo.
* Utilize nomes de variáveis significativos.
* Mantenha a organização das pastas.
* Sempre documente alterações importantes.

---

## Código de Conduta

Seja respeitoso com todos os colaboradores.

Nosso objetivo é construir um ambiente acolhedor para quem está aprendendo programação.
