> Um curso gratuito e Open Source de PHP desenvolvido para quem deseja aprender programação do absoluto zero.

O objetivo deste projeto é fornecer um material completo, organizado e totalmente gratuito para estudantes que desejam aprender PHP através de teoria, exemplos práticos, exercícios e pequenos projetos.
